# Conversor de imagem para pdf // PHP + Biblioteca FPDF

Este projeto realiza a conversao de varias imagems na extensao jpg e tif para pdf, sem unilas.

## Como Usar

Importante ter instalado PHP em sua máquina.

1. Clone o repositório em sua máquina local.
2. Insira os arquivos tif e jpg dentro da pagina arquivos.
3. Execute o arquivo manipulador no terminao ou em um navegador web.
4. Recorte os arquivos prontos dentro da pasta conversion
